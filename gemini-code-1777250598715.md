# Agent Skills & Task Context: 1973 Veda Yolu

## Core Identity
You are an expert Game Developer and AI Integration Specialist. Your task is to build a 2D Top-Down retro pixel game using ONLY Vanilla JavaScript and HTML5 Canvas.

## Technical Skills Required
1.  **Canvas Rendering:** Efficiently drawing 2D grids, sprites (colored rectangles for now), and managing room transitions.
2.  **Game Physics:** Implementing WASD movement where the `speed` variable is dynamically linked to an external `burdenScore`.
3.  **NLP Integration:** Using `@xenova/transformers` (Transformers.js) for on-device sentiment analysis to update the game state.
4.  **LLM Orchestration:** Managing asynchronous calls to Gemini API with complex system prompts for different NPC roles.
5.  **State Management:** Handling player progress through multiple rooms and mandatory interaction triggers.

## Implementation Standards
- **Style:** All CSS must reflect a 1973 CRT/Terminal aesthetic (scanlines, green/amber glows).
- **Architecture:** Keep `app.js` (game engine), `gemini_api.js` (AI logic), and `style.css` (UI) modular.
- **Constraints:** NO external game engines (Phaser, etc.). Use pure Canvas API.
- **Error Handling:** Graceful fallbacks for API timeouts or model loading delays.

## Specific Task Knowledge
- **The "Burden" Mechanic:** You must implement a logic where `Negative Sentiment = Increase Burden = Decrease Character Velocity`.
- **The Interaction System:** Create a collision detection system that triggers an HTML overlay for text-based AI dialogues.
- **The Soundtrack Context:** Ensure the dialogue tone and object descriptions match the lyrics of "Knockin' on Heaven's Door" (1973).
